# Bardanko'l YFK — Diplom ishi taqdimoti

**Mavzu:** Jismoniy shaxslar daromadlarini hisobga olish va soliqqa tortish amaliyoti  
**Korxona:** Bardanko'l yo'llardan foydalanish korxonasi

## Ishga tushirish

```bash
npm install
npm run dev
```

## Deploy (Vercel)

```bash
npm run build
# Vercel Dashboard → New Project → Import → bardankol-presentation
```

## Klaviatura boshqaruvi

| Tugma | Funksiya |
|-------|----------|
| `→` yoki `Space` | Keyingi slayd |
| `←` | Oldingi slayd |
| `F` | To'liq ekran |
| `G` | Slaydlar paneli |
| `Home` | Birinchi slayd |
| `End` | Oxirgi slayd |

## Slaydlar (14 ta)

1. Titul slayd
2. Mavzuning dolzarbligi
3. Maqsad va vazifalar *(majburiy)*
4. Ob'ekt, predmet va metodlar *(majburiy)*
5. Korxona iqtisodiy tavsifi
6. Moliyaviy tahlil (2023–2025)
7. Huquqiy asoslar (SK 364–401)
8. Xorijiy tajriba (AQSh, Germaniya, Rossiya, MDH)
9. JSHDS hisoblash tartibi + misol
10. Hisobot va nazorat tizimi
11. Daromadlar tarkibi va soliq bazasi
12. Aniqlangan muammolar
13. Takomillashtirish yo'nalishlari (7 ta)
14. Xulosa va tavsiyalar
